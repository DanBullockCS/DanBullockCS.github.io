KeepUp!

KeepUp! is a simple Android game made in Unity. In it, you keep a paper ball up as long as you can by bouncing it up with an eraser. It was the first android game I made in Unity so it was a good learning experience leading to greater projects. The project is free and located here.

ClassroomCompanion

ClassroomCompanion is my undergraduate design thesis. It is a website built to assist teachers with various mundane teaching tasks. It is a node.js project using MongoDB for the backend. The website's design process gave me some time to hone my JavaScript skills especially. The project features an Attendance, Grade, and Performance Tracker for students; the code is posted here.

League Kill Viewer

The League Kill Viewer is a D3 dashboard I made with Kevin Desousa and Danooshan Sureshkumar. It takes League of Legends professional game data from 2015 to 2018 and displays it using various visualizations. This was done for the Information Visualization course at Ontario Tech. The project can be found here and is live at to view on my website here.
